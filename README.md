# RC Manager Cloud 4.2 — Estoque Inteligente

Inclui:
- cadastro de produto;
- código, categoria, tamanho e cor;
- preço de custo e venda;
- quantidade e estoque mínimo;
- foto por link;
- busca rápida;
- edição e exclusão;
- entrada e saída de estoque;
- alerta de estoque baixo;
- indicadores de quantidade e valor estimado;
- sincronização em tempo real pelo Firestore.

Observação: o envio direto de fotos para o Firebase Storage será adicionado em uma próxima atualização. Nesta versão, a foto pode ser informada por link.
